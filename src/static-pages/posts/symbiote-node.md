# Symbiote Node

Archived package workspace for earlier RND-PRO Symbiote experiments: integration glue, migration tooling, package organization, and early links between UI primitives and execution-engine prototypes.

It is useful historical context, but it should not be read as the current center of the Symbiote line. The active project connection is Symbiote Workspace, Symbiote UI, and Symbiote Engine.
